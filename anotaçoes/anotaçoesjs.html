Em JavaScript, existem três tipos principais de estruturas de repetição: for, while e do-while. while =pode se chamar ela de (enquanto) é uma estrutura de repetição que executa um bloco de código enquanto uma condição for verdadeira. A condição é verificada antes de cada iteração, e se for falsa, o loop é encerrado. do-while = é uma estrutura de repetição que executa um bloco de código pelo menos uma vez, e depois continua a executar enquanto uma condição for verdadeira. A condição é verificada após cada iteração, garantindo que o bloco de código seja executado pelo menos uma vez. for = é uma estrutura de repetição que permite iterar sobre uma sequência de valores, como um array ou um intervalo numérico. É especialmente útil quando se sabe o número exato de iterações que se deseja realizar. for (inicialização; condição; incremento) { // bloco de código a ser executado } inicialização = é a parte do loop onde você define uma variável de controle, que geralmente é usada para contar o número de iterações. condição = é a parte do loop que verifica se a variável de controle atende a um critério específico. Enquanto essa condição for verdadeira, o loop continuará a executar. incremento = é a parte do loop que atualiza a variável de controle após cada iteração.
