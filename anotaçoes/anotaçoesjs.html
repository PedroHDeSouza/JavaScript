-------------------------------------------------------------------- modulo A: window.alert('') = usado para aparecer alertas na aba do navegador window.confirm('') = usado para interagir com usuario com confirmaçao ou negaçao window.prompt('') = usado para interagir com o usuario, com perguntas --------------------------------------------------------------------- modulo B: var/let = comando para variaveis, pode se usar qualquer um dos dois "=" = simbolo de igual unitario, significa recebe data types: numbers = numeros em geral, tanto inteiro quanto reais string = sao cadeias de caracteres boolean = sao usado para "true" e "false" null = ausência intencional de qualquer valor de objeto undefined = significa que uma variável foi declarada, mas não tem um valor atribuído object = refere-se a uma coleção de propriedades e métodos, onde cada propriedade é uma associação entre uma chave (ou nome) e um valor function = significa que estamos a definir uma função, que é um bloco de código que executa uma tarefa específica e pode ser chamada múltiplas vezes
